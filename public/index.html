🚀 FinTrack API

API de gestion financière personnelle

Documentation Swagger : http://localhost:3000/api

📋 Endpoints principaux

POST /auth/register - Créer un compte utilisateur
POST /auth/login - Se connecter et obtenir un token JWT
GET /accounts - Liste des comptes avec balance totale
🔐 Authentification requise
GET /accounts/balance/total - Balance générale
🔐 Authentification requise
POST /accounts - Créer un nouveau compte
🔐 Authentification requise
GET /transactions - Liste des transactions
🔐 Authentification requise
POST /transactions - Créer une transaction
🔐 Authentification requise
GET /dashboard - Tableau de bord complet
🔐 Authentification requise
GET /dashboard/categories - Top catégories
🔐 Authentification requise

🔑 Authentification

L'API utilise l'authentification JWT Bearer Token :

  1. Créez un compte avec POST /auth/register
  2. Connectez-vous avec POST /auth/login
  3. Utilisez le token dans l'en-tête : Authorization: Bearer <token>
